# 电缆桥架饱和度仪表盘全面升级 — 设计文档

**日期**: 2026-06-09
**状态**: 已确认

## 目标

将当前简洁的 Streamlit 仪表盘升级为暗色科技风的全面监控仪表盘，包含 6 大功能模块，整体字体放大以提升可读性。

## 视觉风格

- **暗色科技风**：深蓝黑背景 (#0b1121 / #111b30)，荧光蓝/紫/绿配色
- **字体放大**：标题 22px+，数据数字 48px+，正文 14px+（相比当前 Streamlit 默认字体显著放大）
- **卡片布局**：所有模块使用圆角卡片 + 1px 边框分割
- **CSS 注入**：通过 `st.markdown` + `unsafe_allow_html` 注入自定义暗色主题

## 功能模块

### 1. 🎯 环形仪表盘
- SVG 实现环形/半圆形仪表盘
- 三色分级：绿色 0-70%，黄色 70-85%，红色 85%+
- 中心大字显示当前饱和度数值
- 底部颜色图例

### 2. 📊 统计卡片组
- 4 个卡片：实时值、今日平均、最大值(含时间)、最小值(含时间)
- 实时值卡片显示与上次值的变化量（▲/▼）
- 每张卡片大字体数值 + 小字标题

### 3. 🚨 智能报警
- 可设置阈值（默认 85%）
- 超标时报警状态栏变红，环形仪表盘红色闪烁动画
- 统计今日报警次数、累计报警时长
- 阈值设置入口（侧边栏滑块）

### 4. 📏 阈值进度条
- 横向渐变进度条（绿→黄→红）
- 显示当前值占阈值百分比
- 标注 0% / 当前值 / 100%

### 5. 📋 历史数据表格
- 底部展示最近 20 条数据
- 列：序号、时间、饱和度、状态（正常/警告/危险）
- 带颜色标签的状态列

### 6. 💾 数据持久化
- SQLite 本地存储（`data/saturation.db`）
- 启动时加载历史数据
- 支持 CSV 导出（侧边栏按钮）
- 支持数据清空（侧边栏按钮）

## 侧边栏控制

- 刷新频率滑块（1-10 秒）
- 报警阈值滑块（70%-95%）
- 开始/停止监控复选框
- CSV 导出按钮
- 数据清空按钮

## 技术实现

- **单文件** `dashboard.py`，基于 Streamlit
- **CSS** 内联在 `st.markdown` 中注入
- **环形图**：SVG circle + stroke-dasharray 实现
- **数据存储**：Python sqlite3 标准库
- **CSV 导出**：pandas `to_csv` + base64 下载链接
- **依赖**：streamlit, requests, pandas, sqlite3（均为标准库或已安装）

## 非功能需求

- 页面不闪烁（使用 placeholder 增量更新）
- 字体整体放大，适合监控大屏远距离查看
- 刷新时保持历史数据连续性
- SQLite 文件自动创建，无额外配置
